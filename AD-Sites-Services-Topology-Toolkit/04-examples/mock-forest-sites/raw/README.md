# Mock Raw Exports

This folder contains a mock raw AD Sites and Services collection JSON for offline converter and renderer testing.

The mock dataset includes:

- Five AD sites.
- Seven subnet rows, including one unassigned subnet warning case.
- Four raw site links expanded into five deduplicated site-pair links.
- One site link containing more than two sites.
- One site with no domain controller.
- Two global catalog domain controllers.
- One RODC.
- Three configured replication connection objects.
- Four observed replication partner metadata rows.
- One replication failure.
- One replication queue/backlog indicator.
- Eight replication topology evidence rows that distinguish configured connection evidence from observed metadata and failures.
